Esta é uma recriação interativa do inventário de The Legend of Zelda: The Wind Waker HD desenvolvida com Blazor WebAssembly. O projeto simula a interface do original do jogo.

Acesse através desse link: <https://dhavirodrigues.github.io/ZWWHDII/>

Este é um projeto não oficial, feito por fã, apenas para fins de estudo.
Todos os recursos relacionados a **The Legend of Zelda: The Wind Waker** são propriedade exclusiva da **Nintendo**.
